# Jogo da Memória em Python

Este projeto é um jogo da memória desenvolvido em Python utilizando a biblioteca Tkinter para criar a interface gráfica.

O objetivo do jogo é encontrar todos os pares de cartas iguais. Cada partida utiliza um conjunto diferente de emojis, escolhido aleatoriamente.

---

## Funcionalidades

O jogo possui as seguintes funcionalidades:

- Interface gráfica construída com Tkinter
- Cartas representadas por emojis
- Conjuntos de emojis escolhidos aleatoriamente a cada partida
- Cartas embaralhadas aleatoriamente
- Sistema de comparação de pares
- Contador de tentativas do jogador
- Destaque visual para pares encontrados
- Mensagem de vitória ao completar o jogo
- Botão para reiniciar a partida
- Cartas redimensionáveis conforme o tamanho da janela

---

## Tecnologias Utilizadas

- Python
- Tkinter
- Random

Tkinter foi utilizado para construir a interface gráfica do jogo e Random para gerar escolhas aleatórias das cartas.

---

## Como executar o projeto

1. Certifique-se de que o Python 3 está instalado em seu computador.
2. Abra a pasta do projeto em uma IDE como o PyCharm.
3. Execute o arquivo `main.py`.

O jogo será aberto em uma janela gráfica.
